# OnlineLearn 在线学习平台 - 启动文档

## 📁 项目结构

```
OnlineLearn/
├── OnlineLearnApi - idea/    # 后端（Spring Boot + MyBatis-Plus + MySQL）
└── OnlineLearnVue/           # 前端（Vue 2 + Element UI）
```

---

## 一、环境要求

| 工具 | 版本要求 |
|------|---------|
| JDK | **1.8**（必须 JDK 8，项目基于 Java 8 构建） |
| Maven | 3.x |
| MySQL | 5.7+ / 8.0 |
| Node.js | 12.x+ |
| npm | 6.x+ |

## 二、启动后端 (OnlineLearnApi)

### 2.1 创建数据库

1. 登录 MySQL：
```bash
mysql -u root -p
```

2. 创建数据库：
```sql
CREATE DATABASE IF NOT EXISTS online_learn DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

3. 退出 MySQL，导入数据表结构和初始数据：
```bash
mysql -u root -p online_learn < "OnlineLearnApi - idea/online_learn.sql"
```

### 2.2 配置数据库连接

打开 `OnlineLearnApi - idea/src/main/resources/application.yml`，根据本地 MySQL 配置修改：

```yaml
server:
  port: 9251

spring:
  datasource:
    driverClassName: com.mysql.cj.jdbc.Driver
    url: jdbc:mysql://localhost:3306/online_learn?&useSSL=false&allowPublicKeyRetrieval=true&serverTimezone=Asia/Shanghai
    username: root          # 修改为你的数据库用户名
    password: 123456        # 修改为你的数据库密码
```

### 2.3 启动后端服务

**方式一：Maven 直接启动**
```bash
cd "OnlineLearnApi - idea"
mvn spring-boot:run
```

**方式二：打包后运行**
```bash
cd "OnlineLearnApi - idea"
mvn clean package -DskipTests
java -jar target/OnlineLearnApi-0.0.1-SNAPSHOT.jar
```

启动成功后访问：http://localhost:9251

## 三、启动前端 (OnlineLearnVue)

### 3.1 安装依赖

```bash
cd OnlineLearnVue
npm install --legacy-peer-deps
```

> `--legacy-peer-deps` 是因为 `less-loader@10.2.0` 依赖 Webpack 5，但 Vue CLI 4.x 使用 Webpack 4，需忽略 peer 依赖冲突。

### 3.2 启动开发服务器

```bash
# Windows PowerShell（Node.js >= 17 需设置 OpenSSL 兼容）
$env:NODE_OPTIONS="--openssl-legacy-provider"
npm run serve
```

```bash
# Linux / macOS（Node.js >= 17 需设置 OpenSSL 兼容）
export NODE_OPTIONS=--openssl-legacy-provider
npm run serve
```

> 如果使用 **Node.js < 17**，则无需设置 `NODE_OPTIONS`，直接 `npm run serve` 即可。

启动成功后访问：**http://localhost:9252**

## 五、常见问题

### 5.1 端口被占用
- 后端端口 `9251` 被占用：修改 `application.yml` 中的 `server.port`
- 前端端口 `9252` 被占用：修改 `vue.config.js` 中的 `devServer.port`

### 5.2 MySQL 连接失败
- 检查 MySQL 服务是否运行
- 检查 `application.yml` 中的用户名和密码是否正确
- 检查数据库 `online_learn` 是否已创建

### 5.3 Maven 依赖下载慢
项目已配置阿里云镜像仓库（见 `pom.xml`），如仍有问题可检查网络连接。

### 5.4 npm 安装依赖慢
可配置淘宝镜像：
```bash
npm config set registry https://registry.npmmirror.com
```

### 5.5 Node.js >= 17 启动报错 `error:0308010C:digital envelope routines::unsupported`
设置环境变量 `NODE_OPTIONS=--openssl-legacy-provider` 即可解决。

### 5.6 前端代理跨域配置
前端 `vue.config.js` 已配置代理，将 `/api` 请求转发到后端 `http://localhost:9251`：
```javascript
proxy: {
  '/api': {
    target: 'http://localhost:9251/',
    changeOrigin: true,
    pathRewrite: { '^/api': '' }
  }
}
```

## 六、完整启动流程

```powershell
# ===== 1. 启动后端（需安装 JDK 8 + Maven + MySQL） =====
# 1a. 创建数据库并导入数据
mysql -u root -p -e "CREATE DATABASE IF NOT EXISTS online_learn DEFAULT CHARACTER SET utf8mb4;"
mysql -u root -p online_learn < "OnlineLearnApi - idea/online_learn.sql"

# 1b. 启动后端
cd "OnlineLearnApi - idea"; mvn spring-boot:run

# ===== 2. 新开终端，启动前端 =====
cd OnlineLearnVue
npm install --legacy-peer-deps
$env:NODE_OPTIONS="--openssl-legacy-provider"
npm run serve
```
