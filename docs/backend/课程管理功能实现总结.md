# 教师端课程管理功能实现总结

## 概述

本次实现了教师端的课程管理功能，包含课程列表页和课程大纲编辑器两大核心模块。课程列表页展示当前教师的所有课程，支持搜索、新建、删除；课程大纲编辑器提供章节树管理、类型切换、内容配置等完整功能。

---

## 一、新增文件

### 1. `frontend/src/views/teacher/TeacherCourseList.vue`

**课程列表页**，功能包括：
- 卡片式网格布局展示课程
- 搜索过滤（按课程名称模糊匹配）
- 新建课程对话框（表单验证）
- 删除课程确认弹窗
- 点击课程跳转到编辑器页面
- 状态标签（已发布/草稿）

### 2. `frontend/src/views/teacher/TeacherCourseManagement.vue`

**课程大纲编辑器**，功能包括：

#### 左侧：章节树大纲面板
- 多级树形结构（无限层级）
- 拖拽排序（同级上下移动）
- 拖拽调整层级（拖到其他节点上作为子节点）
- 每行显示：章节标题、类型图标（视频/测验/阅读）、状态圆点
- 操作按钮（添加子章节、复制、删除、折叠/展开）
- 键盘快捷键：Enter（同级新建）、Tab（缩进为子级）、Shift+Tab（提升层级）、Delete（删除）
- 章节缺失资料时显示橙色警告点

#### 右侧：配置面板
- 顶部显示当前选中章节的路径和标题
- **基础信息**：标题输入、简介输入
- **类型切换**：分段控制器（视频/测验/阅读），切换时保留通用内容
- **智能资料配置**：
  - 视频类型：视频链接输入
  - 测验类型：题目数量、及格分数配置
  - 阅读类型：富文本编辑器（使用 Quill）
- **发布设置**：可见性开关

## 三、技术要点

### 后端交互格式

所有 API 调用使用 axios，后端响应格式统一为：

```json
{
  "code": 200,
  "resultData": [...]  // 或单个对象
}
```

具体接口：
- `GET /study/teacher/course/myCourses?userId={id}` — 获取教师课程列表
- `GET /study/teacher/chapter/getChapters/{courseId}` — 获取章节树
- `POST /study/teacher/chapter/saveChapters` — 保存章节树（body: `{ courseId, chapters }`）
- `POST /study/teacher/course/create` — 创建课程（body: `{ courseName, description, userId }`）
- `DELETE /study/teacher/course/delete/{id}` — 删除课程

### 前端章节数据结构

```javascript
// 章节树结构（递归）
{
  id: string,            // 唯一标识
  title: string,         // 章节标题
  type: string,          // 'video' | 'quiz' | 'reading'
  description: string,   // 简介
  visible: boolean,      // 是否可见
  publishStatus: string, // 'draft' | 'published'
  content: {             // 根据类型不同
    videoUrl: '',        // 视频链接
    questionCount: 0,    // 测验题目数
    passScore: 60,       // 及格分数
    readingContent: ''   // 阅读内容（HTML）
  },
  children: []           // 子章节数组
}
```

### 用户身份获取

```javascript
const userId = this.$store?.state?.user?.id || localStorage.getItem('userId')
```

## 四、修复问题

### 问题：`getMyCourses` 传参格式错误

**现象**：仪表盘和课程列表显示全部课程而非当前教师课程

**原因**：`getMyCourses(userId)` 直接传数字，axios 将数字作为 URL 参数拼接，后端未正确识别 `userId` 参数

**修复**：改为 `getMyCourses({ userId })` 传对象，axios 将其编码为 `?userId=2` 正确传递

### 问题：路由名称不匹配

**现象**：课程列表点击后无法跳转到编辑器

**修复**：将路由 `name` 从 `TeacherChapterEditor` 改为 `TeacherCourseManagement`，保持与组件名一致

## 五、使用说明

1. 教师登录后，侧边栏出现「课程管理」入口
2. 点击进入课程列表，可看到当前教师创建的所有课程
3. 点击「新建课程」创建新课程
4. 点击课程卡片进入大纲编辑器
5. 在左侧章节树中点击「+」添加章节，拖拽调整顺序/层级
6. 在右侧配置面板中设置章节标题、类型、内容
7. 点击「保存」持久化章节数据
