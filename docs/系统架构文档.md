# OnlineLearn 在线学习平台 — 系统架构文档

> 版本: v1.0  
> 最后更新: 2026-05-02

---

## 1. 项目概述

OnlineLearn 是一个面向在线学习场景的 Web 应用平台，支持**管理员、教师、学生**三种角色，提供课程管理、作业管理、视频学习、在线练习、问答互动等核心功能。

| 模块 | 技术栈 | 端口 |
|------|--------|------|
| 前端 (OnlineLearnVue) | Vue 2 + Element UI | 9252 |
| 后端 (OnlineLearnApi) | Spring Boot 2.5.6 + MyBatis-Plus | 9251 |
| 数据库 | MySQL 5.7+ / 8.0 | 3306 |

### 4.3 路由与权限

采用 **Vue Router 导航守卫** (`router.beforeEach`)，根据 Cookie 中 `userId` 判断登录状态，未登录时重定向到登录页。

### 4.4 关于跨域前端代理

前端通过 `vue.config.js` 中的 `devServer.proxy`，将 `/api` 请求代理转发到后端 `http://localhost:9251`，解决开发环境跨域问题。

## 6. 关键设计决策

### 6.1 为什么使用 MyBatis-Plus 而非 JPA？
- 项目包含大量多表关联查询和复杂统计（作业成绩、观看时长聚合），MyBatis-Plus 的 XML 映射更灵活
- MyBatis-Plus 的分页插件 `PaginationInterceptor` 简化分页实现

### 6.2 为什么前端使用 Vue 2 而非 Vue 3？
- 项目启动时 Vue 3 生态尚不成熟，Element UI 只支持 Vue 2
- mavon-editor、vue-video-player 等依赖均基于 Vue 2

### 6.3 关于文件上传
- `application.yml` 中 `multipart.max-file-size` 设为 **1024GB**（实际受硬件限制），用于视频上传
- 后端 `FileUtil.java` 处理文件存储

### 6.4 关于 Swagger
- 使用 springfox 2.9.2 自动生成 API 文档
- 访问地址: `http://localhost:9251/swagger-ui.html`

## 7. 部署架构

```
[开发环境]
┌─────────────────────────────────────┐
│  本地 PC                              │
│  ┌─────────┐   ┌─────────┐          │
│  │ Frontend│──▶│ Backend │──▶ MySQL │
│  │ :9252   │   │ :9251   │   :3306  │

生产环境建议：
- 前端打包为静态文件，部署至 Nginx
- 后端打包为 JAR，部署至服务器
- MySQL 单独部署或使用云数据库
